## Overview
FinPlay is a pirate-themed, gamified financial planning web app that makes managing personal finances engaging and fun. Users navigate a pirate-inspired UI to securely manage savings, track transactions, and access financial tools with interactive animations and responsive design.

## Features
- Pirate-style authentication with animated ocean and treasure visuals
- Real-time validation and interactive UI elements for login and registration
- Responsive design optimized for mobile and desktop
- Backend integration for transaction management and payment processing
- Gamified UI components including celebratory animations for user actions

## Technologies Used
- React.js for frontend components and state management
- CSS3 with custom animations and responsive layouts
- Node.js/Express for backend API integration
- Font Awesome icons for thematic design elements

## Getting Started
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Start the development server with `npm start`.
4. Can also access the app from deplyment section.

## Usage
- Register a new pirate captain account to begin managing your financial treasures.
- Log in as an existing captain to view and manage your financial data.
- Enjoy interactive UI feedback and celebrate your financial milestones.
